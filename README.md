# Github API - React Native

ReactNative application developed during BootCamp GoStack 7 that uses the github API to list and open a webview of the starred repositories by users
# Setup

### Installation

- run `npm install` or `yarn install`

### Run on Device

- run `yarn android` / `yarn ios` to run on device/emulator

or
- run `npm run android` / `npm run ios` to run on device/emulator
